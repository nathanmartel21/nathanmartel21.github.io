# nathanmartel21.github.io

Personal website of Nathan Martel with four pages: Home, About, Blog, and Contact.

Included pages:
- `index.html`: home page with presentation and site links.
- `about.html`: about section with skills and interests.
- `blog.html`: article cards for technical topics.
- `contact.html`: LinkedIn and email contact links.

Global styles are defined in `style.css`.
